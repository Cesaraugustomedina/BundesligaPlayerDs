# Bundesliga

El conjunto de datos «Jugadores de la Bundesliga» ofrece una recopilación exhaustiva de información sobre todos los jugadores de la liga de fútbol alemana.

 Desde porteros de renombre hasta defensas de gran talento, este conjunto de datos ofrece una amplia gama de información sobre los jugadores, incluyendo sus nombres, nombres completos, edades, estaturas, nacionalidades, lugares de nacimiento, precios, precios máximos, posiciones, números de camiseta, pierna dominante, clubes actuales, fechas de vencimiento de los contratos, fechas de incorporación a los clubes, agentes y patrocinadores.
